# PocketShell

**PocketShell** is a Flutter SSH client with OpenCode agent integration,
built for Android and desktop.

## Highlights

- Full **xterm** terminal with profiles, keys, and snippets
- **OpenCode** agent chat over the connected host API
- Android **home-screen widgets** and deep links
- Background session keep-alive via foreground service
- Light, dark, system, and hacker themes
